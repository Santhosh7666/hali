##  Overview

Hali is a full-stack dashboard application designed to manage users, products, and orders efficiently. It provides role-based access, analytics, and a modern UI for both admin and users.

---

## ✨ Features

*  User Authentication (Login & Register)
*  Role-Based Access Control (Admin / User)
*  Interactive Dashboard & Analytics
*  Product Management System
*  Order Management System
*  Notification System
*  User Profile & Settings

---

##  Tech Stack

###  Frontend

* React.js
* Vite
* CSS

###  Backend

* Node.js
* Express.js

###  Database

* MongoDB

---

##  Project Structure

client/   → Frontend (React Application)
server/   → Backend (Node.js + Express API)

---

### Install Dependencies

cd client
npm install

cd ../server
npm install

### Run the Project

Start Backend:
npm run dev

Start Frontend:
npm run dev

---



##  Environment Variables

Create a `.env` file in the server folder and add:

MONGODB_URI=your_mongodb_connection
PORT=5000
